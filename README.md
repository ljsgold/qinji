## 快速开始

### 方法一：双击启动（推荐）

双击运行 `启动系统.bat` 文件即可自动启动。

### 方法二：命令行启动

```bash
# 安装依赖
pip install -r requirements.txt

# 启动应用
streamlit run app.py
```

启动后，浏览器会自动打开 http://localhost:8501
